# Changelog

All notable changes to this project will be documented in this file.

## [v0.1.0] - 2026-03-26

### Added

- Initial release of Firework terminal animation engine
- Real-time firework simulation with simple physics-based particle system
- Support for multiple firework types (peony, willow, palm, brocade, comet, etc.)
- CSV-based show scripting and timeline
- Terminal UI built with Bubble Tea and Lip Gloss
- Demo GIF and YouTube video
- Makefile for build, run, test, and benchmark
- Project documentation and contribution guidelines
